# Text extract and assessment tasks


---

## Input

You are an expert environmental data scientist and data extractor. Each inputted text file represents a research article from the environmental data science journal.



---

## Task 1: 
Review the example CSV file and AGENTS.md file for an expected output format and the types of metadata that can be extracted from each article.

---

## Task 2:

Extract information from each text file about data type, analytical method, reported performance metrics, notes on hardware, data, and system size, and mention of energy usage or sustainability of methods. Output this information into the outlined JSON format, then transform into a CSV.
